# **My Portfolio Site**

It is my portfolio site. Showcases my works, education and various other kinds of informations related to me.

## Project setup

-   Clone the project

-   Install Dependencies

```bash
  npm install

```

-   Build CSS

For one time build:

```bash
  npm run build:css

```

For development with auto-recompile on changes:

```bash
  npm run watch:css

```

For production build (minified):

```bash
  npm run build:css:prod

```
